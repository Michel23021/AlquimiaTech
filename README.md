# 🧪 Alquimia Tech - Sistema de Gestión Integral

**Alquimia Tech** es una solución de software diseñada para optimizar la operación de establecimientos Soporte Técnico de equipos de cómputo. Este proyecto integra la gestión de inventarios, control de reparaciones y un ticket. 

---

## 👤 Información del Proyecto
* **Autor:** Karen Michel Olaya Doncel - Aprendiz
* **Estado:** En Desarrollo 🛠️
* **Enfoque Visual:** Colorimetria (Morado, Azul, Negro y Dorado)

---

## 🚀 Módulos del Sistema

### 🛠️ Gestión de Soporte Técnico
Permite el rastreo completo de una reparación desde el ingreso hasta la entrega final.
* **Recepción:** Registro de equipos y apertura de tickets de servicio.
* **Taller:** Diagnóstico técnico y actualización de estados en tiempo real.
* **Historial:** Bitácora de cambios para cada dispositivo.


---

## 💻 Tecnologías Utilizadas
* **Base de Datos:** MySQL (Modelo Relacional)
* **Frontend:** HTML5, CSS3 (Efectos de transparencia y desenfoque)
* **Lógica:** JavaScript para manejo de formularios dinámicos.

---

## 📐 Estructura de la Base de Datos
El sistema se basa en un modelo relacional que vincula:
1. **Tickets** con **Equipos** y **Usuarios**.
2. **Ventas** con **Detalles de Producto** y **Pagos**.
3. **Mantenimientos** con técnicos responsables.
